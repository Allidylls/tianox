Boot Mechanism

Tian has a mechanism to start applicatons automatically and load required resources sequentially.

There are two steps for doing this:

1, Add a data-boot attribute to the script tag which src is the main Tian script file, eg:

    <script src=tian.js data-boot=boot.js></script>

2, Load required resources and run the applicaton in the scirpt file referenced by data-boot, eg:

    Tian.require(['a.js', 'b.js', 'c.js'], function() {
        Tian.require(['x.js', 'y.js'], function() {
           var app = new MyApp();
           app.run();
        });
    });

That will do!
