Tian

Global namespace contains all constants, properties, methods, namespaces and classes.

Tian.VERSION

Constant: string

The version number string of Tian.

Tian.ISGECKO

Constant: boolean

True if the userAgent reports the browser to use the Gecko engine.

Tian.BROWSER

Constant: string

A substring of the navigator.userAgent property. Depending on the userAgent property, this will be the empty string or one of the following:

  • "firefox" as Firefox
  • "mozilla" as Mozilla
  • "msie" as Internet Explorer
  • "opera" as Opera
  • "safari" as Safari

Tian.createUniqueID(prefix)

Static method

Parameters:
prefix - string, prefix of the id, defaults to "txID_"

Returns: string

Create a unique identifier for this session. Each time this function is called, a counter is incremented. The return will be the optional prefix (defaults to "txID_") appended with the counter value.

Tian.getScriptLocation(file, doc)

Static method

Parameters:
file - string, script file name, defaults to the file which contains this function
doc - document, context document, defaults to current document

Returns: string

Returns the script file location. The doc is a context document. If file not set, returns the location of Tian script file.

Tian.getElement(args)

Static method

Parameters:
args - string | array | DOMElement

Returns: DOMElement | array

Returns DOMElement(s) of given id(s), DOMElement(s).

Tian.getStyle(element, style)

Static method

Parameters:
element - string | DOMElement, element id or element object
style - string, style name

Returns: string

Returns the style value of an element.

Tian.getViewportElement()

Static method

Parameters:

Returns: DOMElement

Returns die viewport element of the document. The viewport element is usually document.documentElement, except in IE, where it is either document.body or document.documentElement, depending on the document's compatibility mode (see http://code.google.com/p/doctype/wiki/ArticleClientViewportElement).

Tian.pagePosition()

Static method

Parameters:

Returns: array

Calculates the position of an element on the page. Returns a two item array, Left value then Top value. (see http://code.google.com/p/doctype/wiki/ArticlePageOffset)

Tian.extend(dest, source)

Static method

Parameters:
dest - object, destination object
source - object, source object

Returns: object

Copy all properties of a source object to a destination object. Modifies the passed in destination object. Any properties on the source object that are set to undefined will not be (re)set on the destination object. Returns the destination object.

Tian.applyDefaults(to, from)

Static method

Parameters:
to - object
from - object

Returns: object

Takes an object and copies any properties that don't exist from another properties, by analogy with Tian.extend. Returns a reference to the to object. Note that the to argument is modified in place and returned by this function.

Tian.inherit(C, P)

Static method

Parameters:
C - class, the class that inherits
P - class, the superclass to inherit from

Returns: undefined

In addition to the mandatory C and P parameters, an arbitrary number of objects can be passed, which will extend C.

Tian.md5(value, key, raw)

Static method

Parameters:
value - string
key - string, optional key string
raw - boolean, optional, calculate raw MD5 hash or not

Returns: string

JavaScript MD5 implementation. Examples:

Calculate the (hex-encoded) MD5 hash of a given string value
  var hash = Tian.md5("value")

Calculate the (hex-encoded) HMAC-MD5 hash of a given string value and key
  var hash = Tian.md5("value", "key")

Calculate the raw MD5 hash of a given string value
  var hash = Tian.md5("value", null, true)

Calculate the raw HMAC-MD5 hash of a given string value
  var hash = Tian.md5("value", "key", true)

Tian.require(urls, callback, doc)

Static method

Parameters:
urls - string | array, script file URL(s)
callback - function, optional
doc - document, optional context, defaults to current document

Returns: undefined

Loads scirpt file(s) into doc context. The callback will be executed after all files loaded. If a doc is not set, then load all files into current document.

Tian.requireCSS(url, callback, doc)

Static method

Parameters:
url - string, stylesheet file URL
callback - function, optional
doc - document, optional context, defaults to current document

Returns: undefined

Loads css file into doc context. The callback will be executed after the file loaded. If a doc is not set, then load file into current document.

Tian.removeCSS(url, callback, doc)

Static method

Parameters:
url - string, stylesheet file URL
callback - function, optional
doc - document, optional context, defaults to current document

Returns: undefined

Removes css file from doc context. The callback will be executed after the file removed. If a doc is not set, then remove file from current document.

Tian.replaceCSS(oldurl, newurl, callback, doc)

Static method

Parameters:
oldurl - string, stylesheet file which will be removed
newurl - string, stylesheet file which will be loaded
callback - function, optional
doc - document, optional context, defaults to current document

Returns: undefined

Replaces css file within doc context, and execute the callback. If a doc is not set, then replaces files in current document.
