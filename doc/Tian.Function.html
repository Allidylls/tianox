Tian.Function

Sub namespace contains convenience functions for function manipulation.

Tian.Function.bind(func, obj)

Static method

Parameters:
func - function
obj - object

Returns: function

Returns a closure with this set to the passed in object. Bind a function to an object. Method to easily create closures with this altered.

Tian.Function.bindAsEventListener(func, obj)

Static method

Parameters:
func - function
obj - object

Returns: function

Bind a function to an object, and configure it to receive the event object as first parameter when called.

Tian.Function.False()

Static method

Parameters:

Returns: boolean

Returns boolean false. A simple function to that just does return false. We use this to avoid attaching anonymous functions to DOM event handlers, which causes "issues" on IE<8.

Tian.Function.True()

Static method

Parameters:

Returns: boolean

Returns boolean true. A simple function to that just does return true. We use this to avoid attaching anonymous functions to DOM event handlers, which causes "issues" on IE<8.

Tian.Function.Void()

Static method

Parameters:

Returns: undefined

A reusable function that returns undefined.
