Tian.Array

Sub namespace contains convenience functions for array manipulation.

Tian.Array.isArray(obj)

Static method

Parameters:
obj - object

Returns: boolean

Returns true if an object is an array, false if it is not.

Tian.Array.indexOf(array, obj)

Static method

Parameters:
array - object
obj - object

Returns: number

Returns the index at, which the first object was found in the array. If not found, returns -1.

Tian.Array.removeItem(array, item)

Static method

Parameters:
array - object
item - object, the item will be removed

Returns: array

Remove an object from an array. Iterates through the array to find the item, then removes it. Returns a reference to the array.

Tian.Array.filter(array, callback, caller)

Static method

Parameters:
array - object
callback - function, filter function
caller - object, optional caller of callback function

Returns: array

Returns an array of elements from the passed in array for which the callback returns true. Provides the functionality of the Array.prototype.filter extension to the ECMA-262 standard. Where available, Array.prototype.filter will be used. Based on well known example from http://developer.mozilla.org/en/Core_JavaScript_1.5_Reference/Global_Objects/Array/filter. The optional caller object to be set as this when the callback is called.
