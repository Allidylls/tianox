Tian.Buffer

Class for handling fixed length octet buffer.

Tian.Buffer(obj, encoding)

Constructor

Parameters:
obj - number | array | string
encoding - string, supported encodings are hex and base64, defaults to hex

Returns: object

Constructs an instance of Tian.Buffer.

Examples:

    var buffer1 = new Tian.Buffer(4);
    var buffer2 = new Tian.Buffer([0xE0, 0x11, 0x0A, 0xFF]);
    var buffer3 = new Tian.Buffer('E0110AFF', 'hex');
