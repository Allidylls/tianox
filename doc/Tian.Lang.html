Tian.Lang

Sub namespace contains internationalization functions.

Tian.Lang.setCode(code)

Static method

Parameters:
code - string, language code

Returns: undefined

Set the language code for string translation. This code is used by the Tian.Lang.translate method. These codes follow the IETF recommendations at http://www.ietf.org/rfc/rfc3066.txt. If no value is set, the browser's language setting will be tested.

Tian.Lang.getCode()

Static method

Parameters:

Returns: string

Returns the current language code.

Tian.Lang.add(code, key, word)

Static method

Parameters:
code - string, language code
key - string, key of word
word - string, word in code

Returns: boolean

Returns whether an item be added into dictionary successfully. The dictionary can be extended (NOT modified) dynamically by calling this method.

Tian.Lang.translate(key)

Static method

Parameters:
key - string, key of word

Returns: string

Returns a internationalized string. Looks up a key from a dictionary based on the current language string. The value of Tian.Lang.getCode will be used to determine the appropriate dictionary. Dictionaries are stored in Tian.Lang[code].

Tian.i18n(key)

Static method

Parameters:
key - string, key of word

Returns: string

Alias of Tian.Lang.translate.
