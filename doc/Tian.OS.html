Tian.OS

The main class of web desktop system, including a title bar, a main panel for starting applications, a window managing system and an event processing system. The style of OSes are themed with style files and can be changed at run time.

Tian.OS(title, themeHost)

Constructor

Parameters:
title - string, system title string
themeHost - string, theme files host domain

Returns: object, instance of this class

.boot()

Instance method

Parameters:

Returns: undefined

Main entry of system. The basic elements such as window system and title bar will be created here. You can create other objects in derived class.

.exit()

Instance method

Parameters:

Returns: undefined

Quit the application and destroy all elements.

.notify(message)

Instance method

Parameters:
message - string, message string

Returns: undefined

By calling this method, the build in event system will broadcast a 'notify' event with a time stamp and given message string to all 'notify' listeners.

.getTheme()

Instance method

Parameters:

Returns: string

Get current theme name used by system.

.changeTheme(theme)

Instance method

Parameters:
theme - string, name of theme

Returns: undefined

Change the system theme.

.setImagePath(path)

Instance method

Parameters:
path - string, optional URL string

Returns: undefined

Reset image path with preset theme name. If the path parameter is valid, it will use this path as image host domain to get theme materials.

.getImagePath()

Instance method

Parameters:

Returns: string, image path of current theme
