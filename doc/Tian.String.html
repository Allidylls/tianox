Tian.String.format(template, context, args)

Static method

Parameters:
template - string
context - object, optional object
args - array, optional arguments

Returns: string

Given a string with tokens in the form ${token}, return a string with tokens replaced with properties from the given context object. Represent a literal ${ by doubling it, e.g. ${${.

A template has the form literal ${token} where the token will be replaced by the value of context["token"]. The context parameter is an optional object with properties corresponding to the tokens in the format string. If no context is sent, the window object will be used.

The args is an optional arguments to pass to any functions found in the context. If a context property is a function, the token will be replaced by the return from the function called with these arguments.

Tian.String.isNumeric(str)

Static method

Parameters:
str - string

Returns: boolean

Returns whether a string contains only a number.

Examples:

  Tian.String.isNumeric("6.02e23") // true
  Tian.String.isNumeric("12 dozen") // false
  Tian.String.isNumeric("4") // true
  Tian.String.isNumeric(" 4 ") // false

Tian.String.sprintf(format, data)

Static method

Parameters:
format - string, format string
data - string | number, optional data

Returns: string

Javascript implematatioan of C/C++ sprintf.

Examples:

    Tian.String.sprintf("%01.2f", 123.1) // "123.10"
    Tian.String.sprintf("[%10s]", "monkey") // "[ monkey]"
    Tian.String.sprintf("%d", 123456789012345) // "123456789012345"

Tian.String.sscanf(str, format)

Static method

Parameters:
str - string, formatted string with data
format - string

Returns: array

Javascript implematatioan of C/C++ sscanf. Returns an array of data according to the format string.

Examples:

    Tian.String.sscanf("ABC/235001", "ABC/%d") // [235001]
    Tian.String.sscanf("10--20", "%2d--%2d") // [10, 20]
