Tian.Class

The class menufactory. All Tian styled class hierarchies should be constructed by this fundamental function.

Tian.Class(P, proto)

Constructor

Parameters:
P - class, parent class
proto - object, prototype object

Returns: class

Base class used to constructe all other classes, including support for multiple inheritance.

To create a new Tian styled class, use the following syntax:
    var MyClass = Tian.Class(prototype);

To create a new Tian styled class with multiple inheritance, use the following syntax:
    var MyClass = Tian.Class(Class1, Class2, prototype);

Note that instanceof reflection will only reveal Class1 as superclass.

A Tian styled class does have a member function initialize as the constructor which will be called immediately when an instance object of this class is created with new operator.
