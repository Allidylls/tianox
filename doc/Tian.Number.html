Tian.Number

Sub namespace contains convenience functions for number manipulation.

Tian.Number.limitSigDigs(num, sig)

Static method

Parameters:
num - number, float instance of Number
sig - number, number of significant digits

Returns: number

Returns the number, rounded to the specified number of significant digits.

Tian.Number.limitFixDigs(num, fix)

Static method

Parameters:
num - number, float instance of Number
fix - number, number of fixed digits

Returns: number

Returns the number, rounded to the specified number of fixed digits.

Tian.Number.format(num, dec, tsep, dsep)

Static method

Parameters:
num - number, float instance of Number
dec - number, integer of decimal places to round to, defaults to 0
tsep - string, thousands separator, default is ","
dsep - string, decimal separator, default is "."

Returns: string

Returns a string representing the formatted number.

Tian.Number.zeroize(num, length)

Static method

Parameters:
num - number
length - number, length of returned string, defaults to 2

Returns: string

Returns a zero-prefixed string of value which length is specified length or 2 by default.
