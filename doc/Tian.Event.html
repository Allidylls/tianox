Tian.Event

Sub namespace contains utility functions for browser event handling.

Tian.Event.element(event)

Static method

Parameters:
event - object

Returns: DOMElement

Cross browser event element detection. Returns the element that caused the event.

Tian.Event.isSingleTouch(event)

Static method

Parameters:
event - object

Returns: boolean

Determine whether event was caused by a single touch.

Tian.Event.isMultiTouch(event)

Static method

Parameters:
event - object

Returns: boolean

Determine whether event was caused by a multi touch.

Tian.Event.isLeftClick(event)

Static method

Parameters:
event - object

Returns: boolean

Determine whether event was caused by a left mouse click.

Tian.Event.isRightClick(event)

Static method

Parameters:
event - object

Returns: boolean

Determine whether event was caused by a right mouse click.

Tian.Event.stop(event, default)

Static method

Parameters:
event - object
default - boolean, to allow the default browser behaviour, defaults to false

Returns: undefined

Stops an event from propagating. If default true, we stop the event chain but still allow the default browser behaviour (text selection, radio-button clicking, etc).

Tian.Event.findElement(event, tag)

Static method

Parameters:
event - object
tag - string, HTML tag name

Returns: DOMElement

The first node with the given tag, starting from the node the event was triggered on and traversing the DOM upwards.

Tian.Event.observe(element, name, observer, capture)

Static method

Parameters:
event - DOMElement | string, DOM or DOM id
name - string, event name
observer - function, event handler
capture - boolean, use capture or not, defaults to false

Returns: undefined

Starts to observe a DOM element with observer.

Tian.Event.stopObserving(element, name, observer, capture)

Static method

Parameters:
event - DOMElement | string, DOM or DOM id
name - string, event name
observer - function, event handler
capture - boolean, use capture or not, defaults to false

Returns: boolean

Returns whether or not the event observer was removed.

Tian.Event.stopObservingElement(element)

Static method

Parameters:
event - DOMElement | string, DOM or DOM id

Returns: undefined

Given the id of an element to stop observing, cycle through the element's cached observers, calling stopObserving on each one, skipping those entries which can no longer be removed.

Tian.Event.unloadCache()

Static method

Parameters:

Returns: undefined

Cycle through all the element entries in the events cache and call stopObservingElement on each.

Note, for preventing memory leaks in IE, this function will be called on unload event of window by default.
