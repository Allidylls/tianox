Tian.DB

Sub namespace contains convenience functions for local storage manipulation.

Tian.DB.storageAvailable()

Static method

Parameters:

Returns: boolean

Returns true if local storage is available.

Tian.DB.get(key, default)

Static method

Parameters:
key - string, key to look up
default - string, optional default value, if key didn't exist

Returns: string

Returns the key value, default value or null.

Tian.DB.set(key, value)

Static method

Parameters:
key - string, key to set
value - string | number | object

Returns: string

Returns the used value.

Tian.DB.index()

Static method

Parameters:

Returns: array

Returns an array of all used keys.

Tian.DB.deleteKey(key)

Static method

Parameters:
key - string, key to delete

Returns: boolean

Returns true if key existed or false if it didn't.

Tian.DB.flush()

Static method

Parameters:

Returns: boolean

Deletes everything in cache and returns true.
